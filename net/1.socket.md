# socket是什么
socket是计算机之间进行通信的一种约定的方式
起源于linux一切皆文件的哲学
可以用open write/read close的模式操作
是一个特殊的文件
Socket() 返回一个整型的文件描述符

# 本地进程通信
1. 消息传递     管道、消息队列、FIFO
2. 同步         互斥量、条件变量、读写锁、文件和写记录锁、信号量
3. 共享内存     匿名的和具名的
4. rpc

本地进程用pid标识，网络进程怎么标识？
